## 🚀 Quick Start

```bash
pnpm install
pnpm dev
```

Then open the local URL shown by Vite (usually `http://localhost:4173`).

> If you prefer npm:
>
> ```bash
> npm install
> npm run dev
> ```


## ✨ Features

- Smooth, animated page transitions
- Custom portfolio sections: hero, project showcase, skills, contact, process
- Advanced UI primitives from Radix and shadcn/ui-inspired components
- Motion effects with GSAP and Framer Motion
- Responsive layout with Tailwind CSS and custom utility styles
- Light/dark theme support via `next-themes`


## 🔧 Recommended workflow

1. Install dependencies: `pnpm install`
2. Start the app: `pnpm dev`
3. Open `http://localhost:4173`
4. Explore `src/app/components/` to customize sections
5. Edit theme and layout in `src/styles/`

## 🧠 Tech stack

- React 18
- Vite 6
- Tailwind CSS 4
- Framer Motion, GSAP, Three.js
- Radix UI primitives
- pnpm / npm
